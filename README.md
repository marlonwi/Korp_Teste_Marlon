# Sistema de Faturamento

## Tecnologias
- Angular
- .NET (C#)
- Angular Material

## Arquitetura
- Microsserviço de Estoque
- Microsserviço de Faturamento

## Funcionalidades
- Cadastro de produtos
- Criação de notas
- Adição de itens
- Fechamento de nota (baixa de estoque)
- Impressão

## Observações
- Validação de regras de negócio
- Tratamento de erros entre serviços
